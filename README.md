# 💌 ListGo · 让朋友替你做决定

> "今晚我该吃啥？" —— 一个让朋友替你做决定的极简清单工具。

ListGo 不是工作协作工具，是一个**让你感受被关心的小角落**。

## 特性

- 无需注册即可创建清单（匿名）
- 3 种模块：投票 / 待办 / 富文本
- 4 套主题：☀ Day · 🌙 Dark · 🌸 Light Pink · 🥀 Dark Pink
- 一链分享，朋友点开即可参与
- 邀请码制度，保持小圈子氛围

## 本地启动

```bash
npm install
npm run dev
```
